# Co-op Opportunity Recommender & Skill-Gap Analyzer

**Short Description:**  
This is an NLP-powered tool that analyses co-op job postings and a student’s skillset. Then, it recommends the most relevant opportunities for the student and highlights missing skills that the student can learn to enhance their profile.

---

## Project Overview
This project aims to help students find the best-fit co-op roles by:
1. Collecting and preprocessing job postings data.
2. Extracting key skills using NLP.
3. Scoring opportunities based on similarity to a student’s skills.
4. Displaying recommendations and skill-gap insights via a simple web app.

---

## Tech Stack
- **Language(s):** Python  
- **Libraries:** pandas, NumPy, scikit-learn, spaCy, sentence-transformers, Streamlit/FastAPI  
- **Tools:** Git, matplotlib/seaborn for EDA, Streamlit Cloud for hosting
